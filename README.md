## рҹ’Ў WHY I BUILT THIS

I kept seeing people (including myself) eating packaged food with zero idea what's actually inside it. Sure, there's a nutrition label вҖ” but nobody reads it, and even if you do, you don't know if that sodium count is dangerous *for you specifically*.

So I built **NutriScan** вҖ” a full-stack web app that lets you log what you eat, tracks your nutrients in real time, and fires an alert the moment you're approaching or exceeding safe daily limits. Personalized. Data-driven. Actually useful.
